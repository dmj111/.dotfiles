* Emacs startup config
* Using this file
  To try out this file without actually installing it, run:

#+BEGIN_EXAMPLE
  emacs -nw -L . -l init.el
#+END_EXAMPLE

 - =-nw= no window
 - =-L .= add . to search path
 - =-l init.el= run init.el
 - =-Q= (maybe?  quick)
